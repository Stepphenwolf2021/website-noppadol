Layout

The website uses a responsive grid layout with a maximum content width of 65ch (approximately 65 characters) for optimal readability. The main container has a max-width of 75rem (1200px) with responsive padding.

Responsive Breakpoints

  • Desktop: Default (large screens)
  • Tablet: < 1200px (container padding reduced, portfolio grid becomes single column)
  • Mobile: < 900px (navigation becomes mobile menu, profile container becomes column)
  • Small Mobile: < 600px (further spacing reductions)
